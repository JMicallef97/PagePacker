==ABOUT PROJECT==
This project was built using the express edition of Microsoft Visual Studio 2010 C#, targeting .NET Framework 4.0. This project does not have any dependencies asid from .NET framework 4.0.

==PROJECT INTENT==
The intent of this project is to condense a webpage from a series of files spread throughout a folder structure into a single .html file in order to be able to deploy the webpage (an application built using web technologies like HTML, CSS, Javascript and JSON) to any device that supports a browser that will open and display HTML files, such as mobile devices where security restrictions prevent arbitrary access of files from the user's device. The idea behind this design strategy is to leverage the abstraction that HTML and related technologies employ to be able to run in many different browsers and environments, reducing development time and effort needed to port code to different environments and platforms since browsers automatically handle client-specific functions like component render/layout and event handling logic. In addition, applications developed using this approach will be able to seamlessly connect to the internet without requiring separate dependencies, since HTML and related technologies are built around internet connectivity.
